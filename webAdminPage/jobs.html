Jobs

Pro Website Admins aims to employ keen workers with fisrt-hand industry experience in website maintainence, troubleshooting and security. We value you and your work ethics, and our company works with a wide range of employees. We have many job opportunities for interested employees, such as:

Website maintainence

Website mantainece is an important role. A website poorly mantainenced can cause customers to be taken to the wrong pange or potentially lose interest in a company's goal. Help ensure users are not led to "dead" links and that the site functions properly. Report any bugs or problems with a website to website troubleshooting

SALARY: 50k

KEY RESPONSIBILITIES:

  1. Maintaing a website's quality (making sure links lead to correct sections, users aren't taken to wrong webpages, etc)
  2. Reporting any problems with a website to Troubleshooting
  3. Look for ways to make a website more efficient

REQUIRED QUALIFICATIONS

  • 1+ years of Experience in maintainence for websites OR
  • Studying computer science

Website troubleshooting (#r78c2)

Website troubleshooting is similar to maintainence, and is just as important. Here, you will ensure that a website functions correctly by fixing any problems brought up by maintainece. Help ensure there are no issues with code on the website and fix any problems that occur if so

SALARY: 40k

KEY RESPONSIBILITIES:

  1. Fix issues relating to website
  2. Ensure there are no issues with code
  3. Receive feedback from maintainence to check if problems were resolved

REQUIRED QUALIFICATIONS

  • Degree in ANY computer science degree OR
  • 1-2+ years of experience in programming

Website Security (#sr51t)

Security is also an equally important role for PWA. Here, you will ensure that company or customer data is protected, and also defend websites from any potential attacks. Make sure that websites have strong security and that they are well apt and prepared to face any potential cyber attacks

SALARY: 54k

KEY RESPONSIBILITIES:

  1. Protect data kept in website
  2. Defend website from any pontential cyber attacks
  3. Check to see if website has any potential "backdoors" (areas where cyber criminals could access data gathered from website)

REQUIRED QUALIFICATIONS

  • Degree in cybersequirty (masters preferred) OR
  • 1-2+ years experience in cybersecurity
